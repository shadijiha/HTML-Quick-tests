Magic damage
Magic damage is a new form of damage that ignores target's armor. Magic damage can be reduced by and increased generaly by ability power

If magic damage critical strikes, it appears as a purple critical strike icon instead of orange.

Attack speed
Attack speed system has been completly updated and simplified:

  • Previous system used the attack speed as a cooldown in second between 2 attacks. New system uses attack speed as the maximum of attacks a unit can do per second. (We will use s for cooldown in seconds and A/S for attacks per seconds)

    • For example, a unit has 0.667 attack speed (1.5 attacks per second) buys an item that gives +40% attack speed:
    • Previous system: 0.667s - (0.667s * 0.40) = 0.4002s (2.49 A/S). This system didn't make any sense since with only one item a unit can get almost maximum attack speed (2.50 is the cap of attack speed a unit can get).
    • New system: 1.5 A/S + 0.40 = 1.90 A/S (0.526s). The new system is much simpler and does not use the s notation in calculations since it was really buggy and confusing.

    With that being said, Player and monsters attack speed had to be adjusted:
    Player's base attack speed: 1.00s 0.67 A/S (1.50s)
    Monsters' base attack speed: 3.00s 0.33 A/S (3.00s)
